# MakeAzureVM
Make a Debian based Azure VM!

# Prerequisites

1. [Install Azure CLI](https://docs.microsoft.com/en-us/cli/azure/install-azure-cli#install-on-debianubuntu-with-apt-get)

# Instructions

1. make build-setup
2. make build-image  [IMAGE_SIZE=n] [RELEASE=jessie]
